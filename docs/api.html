/users post

post

Creates a new User

post /users

Creates a new User

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • email: required (string)
  • id: (number)
  • created_at: (datetime)
  • updated_at: (datetime)
  • firstname: required (string)
  • lastname: required (string)
  • password: required (string)
  • phone: (string)
  • cover: (string)
  • picture: (string)
  • about: (string)
  • followers: (number)
  • following: (number)
  • positiveRatings: (number)
  • negativeRatings: (number)
  • neutralRatings: (number)
  • vacationMode: (boolean)
  • shippingMethods: (object)
    • name: required (string)
    • descriptionSeller: required (string)
    • descriptionBuyer: required (string)

Example:

{
  "firstname": "Marie",
  "lastname": "Curie",
  "email": "marie@curie.com",
  "password": "asd|@123"
}

HTTP status code 201

User was created succesfully

HTTP status code 400

There is an error on the input given for user creation

Body

Media type: application/json

Type: object

Properties
  • message: required (string)
  • errors: (object)
    • /^[a-z_]*$/: required (array of string)

Example:

{
  "message": "The given data was invalid.",
  "errors": {
    "firstname": [
      "The firstname field is invalid or empty."
    ],
    "email": [
      "The email field is invalid or empty"
    ],
    "exists": [
      "There is another user registered with the same email"
    ],
    "password": [
      "The password field is invalid or empty"
    ]
  }
}

get /users/password/recovery/{email}

Requires a password recovery link for an user

  • Request
  • Response

URI Parameters

  • email: required (string)

HTTP status code 200

A reset link has been send to the email specified

HTTP status code 400

Body

Media type: application/json

Type: object

Properties
  • message: required (string)
  • errors: (object)
    • /^[a-z_]*$/: required (array of string)

Example:

{
  "message": "There was an error on the request",
  "errors": {
    "attempts": [
      "Too many password request for the given email"
    ],
    "email": [
      "The given email is invalid or empty"
    ]
  }
}
